Portfolio

Here is a list of projects I've done.

Mouseless Mouse

I was part of the team for mouseless mouse which was a project to create a mouse cursor device in the form of a glove which you could maneuver around. As part of the mouseless movement team I used linear algebra and statistics in order to calibrate the magnetic sensors so that the device is able to move smoothly depending on how it is oriented. This project was programmed in C++.

UnityStation

I developed code for an open source game UnityStation including a system for rotating different maps used in the game and adding DNA and bloodtypes to entities in the game. For this project I used C# and the Unity Engine to develop code.

Hexapods

Hexapods is a project I will be working on this semester which will focus on making programming various types of movement for six legged robots. In addition, the project will involve deploying multiple robots to conduct search operations.

Prometheus Commodities Exchange

In C# and the Discord .NET library I made a discord bot which would simulate the exchange of different commodities for a game. In the bot I was able to sort how to arrange the data to match buy and sell offers and make the requisite changes to external sheets when sales were made.

National Cyber League

I have participated in National Cyber League, a capture the flags cybersecurity competition which tests cryptography, network analysis, and software exploitation. I used Python, Perl, software Dissasemblers, and javascript for this competition.

The New Order: Last Days of Europe

I was on the development team for The New Order: Last Days of Europe. As part of the development team I designed and implemented new systems in the game. I used Paradox Scripting in my time as a developer for the project.
